# TechStore

TechStore is a sample ecommerce storefront built for software testing practice.

## Tech stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## Getting started

```sh
npm install
npm run dev
```

## Scripts

- `npm run dev` Start the dev server
- `npm run build` Build for production
- `npm run preview` Preview the production build
- `npm run lint` Run lint checks
